Get Cesium context with your component#

useCesium hooks is available with Resium:

import { useCesium } from "resium";
const ExampleComponent = () => {  const { viewer } = useCesium();  return <p>Cesium Viewer object is{viewer ? "" : " not"} provided here.</p>;};
export default ExampleComponent;

In TypeScript:

import { Viewer } from "cesium";import { useCesium } from "resium";
const ExampleComponent = () => {  const { viewer } = useCesium();  return <p>Cesium Viewer object is{viewer ? "" : " not"} provided here.</p>;};
export default ExampleComponent;
