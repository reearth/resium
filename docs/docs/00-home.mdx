# Resium

![Resium](/resium.gif)

Resium is library of React components for 🌍[Cesium](https://cesium.com/)

```jsx
<Viewer full>
  <Entity
    name="tokyo"
    description="test"
    position={Cartesian3.fromDegrees(139.767052, 35.681167, 100)}
  />
</Viewer>
```
